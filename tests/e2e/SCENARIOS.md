## Каркас (shell)

### S-shell-01: приложение загружается и показывает монитор периода
- **Given:** чистый браузер, dev-сервер запущен
- **When:** открыть `/#/`
- **Then:** видны бренд «Теплобиллинг», шапка «МУП «Тепловые сети», алерт «База пуста — начните с демо-данных»
- **Spec:** `tests/e2e/shell.spec.ts` → `test("S-shell-01: загрузка и монитор периода")`
- **Destructive:** нет

### S-shell-02: PGlite инициализируется — карточка БД показывает таблицы
- **Given:** приложение открыто
- **When:** дождаться карточки БД
- **Then:** «Таблиц создано» ≥ 16; движок «PostgreSQL (PGlite) в браузере»
- **Spec:** `tests/e2e/shell.spec.ts` → `test("S-shell-02: PGlite инициализируется")`
- **Destructive:** нет (IndexedDB в эфемерном профиле теста)

### S-shell-03: навигация по разделам синхронна с меню
- **Given:** приложение открыто
- **When:** кликнуть «Ввод показаний» в меню
- **Then:** URL `#/readings`, заголовок раздела, пункт меню выделен
- **Spec:** `tests/e2e/shell.spec.ts` → `test("S-shell-03: навигация по разделам")`
- **Destructive:** нет

### S-shell-04: кнопка демо-данных наполняет базу
- **Given:** база пуста, кнопка активна после инициализации БД
- **When:** нажать «Наполнить демо-данными»
- **Then:** тост «Демо-данные загружены…», тег «демо-данные загружены», счетчик ЛС = 170
- **Spec:** `tests/e2e/shell.spec.ts` → `test("S-shell-04: кнопка демо-данных наполняет базу")`
- **Destructive:** нет (пишет только в эфемерный IndexedDB теста)
