| Инфокрафт: Формула ЖКХ | коробка 15 800 ₽ + раб. места |
| Аутсорсинг РЦ (Биллинг Эксперт и др.) | ~10 ₽/ЛС/мес ≈ 100 т₽/мес на 10к ЛС |

**Buy-vs-build честно**: одному МУП купить готовое в 4–10 раз дешевле заказной разработки (TCO 3 года: SaaS 0,3–1,3 млн ₽; коробка+внедрение 1,5–2,5 млн ₽). Следствие для продукта: **не конкурируем ценой лицензии**. Наши углы: современный UX (у конкурентов — интерфейсы десятилетней давности), скорость внедрения (браузер против «внедрения франчайзи»), теплоспецифика (двухкомпонентный ГВС, юрлица-теплосеть), прозрачность каждой цифры (calc_trace → «объяснить расчет» — ни у кого нет), данные у МУП (PGlite локально).

## 4. Журнал поправок (что адверсариальная проверка опровергла)

1. ~~Корректировка отопления = ф.3(2)~~ → **ф.3(4)**; 3(2) утратила силу с 01.01.2019 (ПП 1708).
2. ~~ф.2 для жилого дома, ф.2(1) для МКД~~ → различие — **способ оплаты** (сезон vs равномерно), не тип здания.
3. ~~ф.3(1) появилась из ПКС №46-П~~ → из **ПКС №30-П от 10.07.2018**; №46-П — про индивидуальные источники тепла.
4. ~~1С:Управление теплосетью 2 ≈ 280 400 ₽~~ → **364 500 ₽** (solutions.1c.ru, 18.07.2026; цены 1С выросли 04–07.2026 на 12–14%).
5. ~~ПП №1871 меняет коэффициенты ГВС~~ → №1871 (от 25.11.2025, в силе с 06.12.2025) касается **только ХВС** (Кпов 1,5→3,0); для ГВС остается 1,5 и только на компонент холодной воды.
6. ~~refine v5 вышел в феврале 2026~~ → 18.09.2025.

Подтверждено: cap пени 9,5% до 01.01.2027 (ПП №329 от 18.03.2025); срок оплаты с 01.03.2026 — 15-е число (177-ФЗ), сдвинуты и сроки платежных документов; ГИС ЖКХ — приказ Минстроя №79/пр от 07.02.2024 (ред. 20.11.2025) заменил 74/114/пр, штраф РСО до 200 000 ₽ (13.19.2 КоАП); QR по ГОСТ обязателен для РСО (п.69 ПП 354), Sum в копейках; Кпов к отоплению отменен (ПП №232).
