# DESIGN.md — Теплобиллинг

Регистр: product (интерфейс служит работе, не витрина). Основа — Ant Design v6, compact-алгоритм, светлая тема.

## Сцена

Оператор расчетного отдела МУП, офис, дневной свет, обычный 22–24" монитор, 6–8 часов в программе ежедневно. Отсюда: светлая тема (принудительно), плотная верстка, минимум движения.

## Цвет (стратегия: Restrained)

| Роль | Значение | Применение |
|---|---|---|
| Primary / акцент | `#0f766e` (глубокий teal) | кнопки действий, выделение в меню, ссылки; ≤10% поверхности |
| Сайдер | `#132538` (тонированный steel-navy) | навигация; текст `#f8fafc`, вторичный `#94a3b8` |
| Фон рабочей области | AntD token (слегка серый) | |
| Поверхности | белый с тенью AntD; шапка `#fbfcfd` | чистые `#fff`/`#000` запрещены |
| Нейтрали | slate-шкала (`#e2e8f0`, `#64748b`, …) | границы, вторичный текст |
| Статусы | AntD success/warning/error | статус периода, валидация показаний, долги |

Запрещено: фиолетовые/неоновые акценты, свечения (glow), градиентный текст, glassmorphism, боковые цветные полосы-акценты толще 1px.

## Типографика

- Системный стек: `-apple-system, 'Segoe UI', Roboto, …` — без внешних CDN-шрифтов (продукт должен работать в офлайн-сети МУП).
- Числа в таблицах и суммах — `font-variant-numeric: tabular-nums`.
- Тире в русских текстах — норма языка, использовать по правилам русской типографики.
- Тело текста ≤ 65ch.

## Плотность и слои

- AntD `compactAlgorithm` глобально; радиус 6.
- Карточки — только для самостоятельных модулей экрана; внутри — разделители и отступы, не вложенные карточки.
- Таблицы данных — главный житель интерфейса: маленький size, липкие заголовки, моноширинные цифры.

## Движение

Минимум: встроенные переходы AntD (hover/active), без появлений-каскадов, без параллаксов. `prefers-reduced-motion` уважается по умолчанию (ничего декоративного нет).

## Состояния (обязательны для каждого экрана)

Загрузка — Skeleton по форме контента; пусто — пояснение «что сделать, чтобы данные появились»; ошибка — Alert с причиной и действием; кнопки во время async — disabled + понятный тост.

## Иконки

Только `@ant-design/icons` (SVG, один стиль Outlined). Эмодзи в интерфейсе запрещены.
